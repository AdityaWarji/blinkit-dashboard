This project is a **Blinkit Sales Analysis Dashboard** created to analyze sales performance, item distribution, outlet performance, and customer ratings using interactive visualizations.

The dashboard helps understand business insights such as total sales, average sales, item categories, outlet types, and rating trends.

---

## Project Overview

The dashboard provides insights into:

* Total Sales Performance
* Average Sales
* Number of Items Sold
* Average Customer Rating
* Outlet Location Analysis
* Outlet Size Distribution
* Item Type Performance
* Fat Content Distribution

This project demonstrates **data visualization, dashboard design, and business insight generation** skills.

---

## Dashboard Features

* Interactive filter panel
* KPI cards for quick insights
* Sales trend visualization
* Outlet performance comparison
* Item category analysis
* Rating distribution insights
